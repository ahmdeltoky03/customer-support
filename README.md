# Customer Support Multi-Agent SystemA multi-agent customer support system built with LangGraph and FastAPI for an e-commerce store.##  Architecture```User Message     ↓[Classifier Agent]     ↓  ┌──┴──┬──────┬──────┐[Order] [Product] [Refund] [General]     ↓[Response]```##  Agents| Agent | Role ||---|---|| **Classifier** | Detects message intent and routes to the right agent || **Order** | Handles order tracking, status, and cancellation || **Product** | Handles product search, stock, and pricing || **Refund** | Handles returns and refund requests || **General** | Handles any other queries |##  Tech Stack- **Backend** — Python, LangGraph, FastAPI- **LLM** — OpenRouter (Claude Sonnet / GPT-4o-mini)- **Frontend** — HTML, Tailwind CSS- **Deploy** — Railway (backend), Vercel (frontend)##  Run Locally### Backend```bashcd backendpip install -r requirements.txtuvicorn main:app --reload```### Frontend```bash# open frontend/index.html in your browser```##  Environment Variables```bashOPENROUTER_API_KEY=sk-or-...```##  Project Structure```customer-support/├── backend/│   ├── agents/│   │   ├── classifier.py│   │   ├── order_agent.py│   │   ├── product_agent.py│   │   ├── refund_agent.py│   │   └── general_agent.py│   ├── graph/│   │   └── support_graph.py│   ├── data/│   │   ├── mock_products.json│   │   ├── mock_orders.json│   │   └── mock_customers.json│   ├── config.py│   ├── tools.py│   └── main.py└── frontend/    └── index.html```
